# Contributing

Thanks for your interest in contributing to the Digital Securities Settlement project.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/digital-securities-settlement.git`
3. Install dependencies: `npm install`
4. Run tests: `npx hardhat test`

## Development

- Solidity contracts are in `contracts/`
- Tests use Hardhat + ethers.js + chai in `test/`
- Deployment scripts are in `scripts/`

## Code Style

- Follow Solidity style guide for smart contracts
- Use NatSpec comments on all public functions
- Write tests for any new functionality
- Run `npx hardhat compile` before submitting

## Pull Requests

1. Create a feature branch from `master`
2. Write clear commit messages
3. Ensure all tests pass
4. Update README if adding new features

## Security

If you discover a security vulnerability, please open an issue or contact the maintainer directly. Do not submit a public PR for security issues.
